# HealthSentinel AI — Frontend Dashboard

AI-Driven Healthcare Anomaly Detection System — React Frontend

## Deploy to Vercel (2 minutes)

### Option A: Vercel CLI
```bash
npm install -g vercel
vercel
```

### Option B: Vercel Dashboard
1. Push this folder to a GitHub repo
2. Go to vercel.com → New Project
3. Import the repo
4. Click Deploy ✅

## Run Locally
```bash
npm install
npm start
```

## Features
- Real-time patient vitals streaming (2s interval)
- Anomaly detection (Autoencoder + Isolation Forest simulation)
- 5 patients across ICU/CCU/General wards
- Live alert feed with severity levels
- Pause/Resume stream
